# 03 User Note

3차시 주제는 Graph Machine Learning입니다.

정리 기준:

- CS의 graph는 각 객체 간 관계성을 나타내는 자료 구조로 설명한다.
- Knowledge Graph는 사람의 지식을 graph로 표현한 것이며, Palantir식 ontology 논의와도 연결된다.
- 사람이 추론할 때 관계성을 가지고 추론하는 경우가 많으므로, 인간의 사고를 따라가는 AI에서는 관계 구조가 중요하다.
- 기계학습 모델은 숫자를 계산하므로, graph의 이산적인 구조를 feature vector로 바꾸는 Graph Representation Learning이 필요하다.
- Graph Embedding, Node Embedding, Graph Neural Network, Graph Convolutional Network를 3차시 핵심 축으로 둔다.
- 3차시 진행 범위는 Graph Representation Learning, GCN 기본 구조, 활용 사례까지로 보고, Knowledge Graph Embedding 상세는 다음 범위로 둔다.

이 파일은 원자료 보관용이며 웹 페이지에는 정제된 차시 노트만 연결한다.
